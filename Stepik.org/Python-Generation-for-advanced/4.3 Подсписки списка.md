# Solution

```python
s = input().split()
list1 = []
for i in range(len(s) + 1):
    for j in range(len(s) + 1):
        list1.append(s[i:j])
list1.sort(key=len)
list1 = list(filter(None, list1))
list1.insert(0, [])
print(list1)
```
