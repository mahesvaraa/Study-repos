```

```

# Solution

```python
numbers = input().split()

result = {}
for num in numbers:
    if num not in result:
        result[num] = 1
        print(num, end=" ")
    else:
        result[num] += 1
        print(num + "_" + str(result[num] - 1), end=" ")

```
