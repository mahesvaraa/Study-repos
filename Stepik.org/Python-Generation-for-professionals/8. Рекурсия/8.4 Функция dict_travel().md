# Solution

```python
def dict_travel(nested_dicts, res=None):
    if res is None: res = []
    for key, value in sorted(nested_dicts.items()):
        if isinstance(value, dict):
            dict_travel(value, res + [key])
        else:
            print('.'.join(res + [key]) + ':', value)
```
